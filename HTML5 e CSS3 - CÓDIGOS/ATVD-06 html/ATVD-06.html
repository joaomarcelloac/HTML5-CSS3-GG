Código-fonte

TAG <code>:

A tag <code> é usada para marcar trechos de código dentro de um conteúdo HTML, indicando que o texto nela contido é código de programação. Ela é uma tag semântica , o que significa que além de formatar o texto (geralmente com uma fonte monoespaçada), ela também comunica aos navegadores e ferramentas (como leitores de tela) que aquele conteúdo é um trecho de código.

Elemento Inline: É um elemento inline, ou seja, ele não quebra a linha automaticamente e pode ser usado dentro de parágrafos ou outros blocos de texto.

Uso Típico: Para trechos curtos de código ou comandos dentro de um texto, por exemplo:

ex: O comando document.getElementById('teste') é escrito em linguagem JavaScript.


TAG <pre>:

A tag <pre> é usada para exibir texto pré-formatado. Isso significa que todo o espaçamento, quebras de linha e formatação presentes no código-fonte serão preservados na renderização da página.

Elemento de Bloco: É um elemento de bloco, ocupando toda a largura disponível e iniciando em uma nova linha.

Uso Típico: Para exibir blocos de código ou texto onde a formatação é importante, por exemplo:

while True:
        num = int(input('Digite um número: '))
        if num % 2 == 0:
            print(f'O número {num} é PAR.')
        else:
            print(f'O número {num} é ÍMPAR.')
        
        continuar = str(input('Deseja continuar? [S/N]: ')).upper().strip()
        if continuar == 'N':
            print('Finalizando o programa...')
            break
        elif continuar != 'S':
            print('Opção inválida! Tente novamente.')

Citações simples

A tag <q> é ideal para trechos pequenos ou específicos que não precisam de um grande destaque visual. Como é um elemento inline, a citação se integra facilmente ao parágrafo sem causar quebras de linha.

ex: Como diria o pai de um amigo meu: o computador é um burro muito rapido

Citações Longas

A tag <blockquote> é utilizada para delimitar uma citação em bloco, ou seja, um trecho de texto que é uma citação extensa de outra fonte. Ela indica que o conteúdo é extraído de outra fonte e, por ser um elemento de bloco, normalmente aparece com uma formatação diferenciada (como recuo) para destacar a citação.

Conforme Alexandre Dumas em O Conde de Monte Cristo:

A vida é uma tempestade (...) Um dia você está tomando sol e no dia seguinte o mar te lança contra as rochas. O que faz de você um homem é o que você faz quando a tempestade vem.

Paramêtro cite na tag <blockquote>

O parâmetro cite dentro da tag <blockquote> é um atributo opcional que serve para indicar a fonte original da citação. Ele não exibe uma URL diretamente na página, mas fornece uma referência semântica para navegadores e mecanismos de busca.

Propósito: Indicar a origem da citação, ajudando na adição do conteúdo.

Semântica: Melhora a acessibilidade e o SEO, pois os mecanismos de busca podem entender melhor a relação entre o conteúdo e sua fonte.

Não Exibido Automaticamente: O usuário não verá uma URL diretamente no navegador, mas ela pode ser utilizada por bots, assistentes e ferramentas de leitura.

ex:

"A persistência é o caminho do êxito." – Charles Chaplin

No edito de código:

<blockquote> cite="https://www.exemplo.com/artigo">

"A persistência é o caminho do êxito." – Charles Chaplin

</blockquote>

Abreviações

A tag <abbr> usada para marcar abreviações e siglas em um documento HTML. Ela melhora a acessibilidade e a experiência do usuário, pois permite exibir o significado da abreviação quando o usuário passa o mouse sobre o texto.

Estou estudando HTML e CSS. Estou adorando!

No edito de código:

<abbr title="HyperText Markup Language">HTML</abbr> e <abbr title="Cascading Style Sheets">CSS</abbr>

Texto invertido

Uma tag <bdo> (Bidirecional Override) é usada em HTML para forçar a direção do texto , independentemente da configuração padrão do navegador ou do idioma do conteúdo.

Texto normal:

Estou aprendendo a criar coisas em HTML.

Texto com <bdo>:

Estou aprendendo a criar coisas em HTML.

Atribuição dir à tag <bdo>

O atributo dir define a direção do texto e aceita dois valores:

ltr (da esquerda para a direita, da esquerda para a direita) → Direção padrão para idiomas como português e inglês.

rtl (direita para esquerda, direita para esquerda) → Usado para idiomas como árabe e hebraico.
