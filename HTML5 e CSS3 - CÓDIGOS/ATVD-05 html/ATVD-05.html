Principais formatações

Negrito / Destaque

Negrito usando a tag <b>: termo em negrito (não semântica, apenas muda o estilo visual do texto para preto)

Negrito usando a tag <strong>: termo em negrito (indica que o texto tem grande importância semântica)


Itálico / Ênfase

Itálico usando a tag <i>: termo em itálico (não semântica, apenas muda o estilo visual do texto)

Itálico usando a tag <em>: termo em itálico (indica que o texto tem grande importância semântica)


Texto marcado

É possivel criar um texto marcado usando a tag <mark>: termo para exemplo de marcação.


Texto grande e pequeno

Criando um texto grande e um texto pequeno neste parágrafo

Tags usadas: <big> e <small>


Texto deletado

Usando a tag <del> é possivel marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.


Texto inserido

Usando a tag <ins> é possivel marcar um texto como inserido para dar uma énfase e indicar que ele foi adicionado depois(indica que o texto tem importância semântica).

Existe também a tag <u>: que também serve para sublinhar o texto. (não semântica).


Texto sobrescrito

Use a tag <sup> para inserir coisas como: x20+3


Texto subscrito

Use a tag <sub> para inserir coisas do tipo: H2O
