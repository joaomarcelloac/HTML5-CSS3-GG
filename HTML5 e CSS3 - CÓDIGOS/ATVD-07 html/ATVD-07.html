Trabalhando com listas

Listas Ordenadas

<ol>: Representa uma lista numerada ou com uma ordem específica (ordered list - Lista Ordenada).

<li>: Os itens da lista são representados por <li> (List Item - Item de Lista).

Usa números, letras ou algarismos romanos.

Os itens seguem uma sequência lógica.

Listas simples:

ex:

  1. Uva
  2. Banana
  3. Maça

Código no editor:

    <ol>
        <li>Uva</li>
        <li>Banana</li>
        <li>Maça</li>
    </ol>

O atributo type define o formato da numeração em listas ordenadas (<ol>). Por padrão, os itens são numerados como 1, 2, 3... , mas podemos modificar isso usando type.

Lista numérica padrão (type="1")

<ol type="1">
  1. Item 1
  2. Item 2
  3. Item 3

Código no editor:

    <ol type="1">
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
    </ol>

O parâmetro start nas listas ordenadas (<ol>)

O atributo start define o número ou letra inicial da contagem em uma lista ordenada (<ol>).

Lista numérica começando do 5

<ol start="5">
  1. Item 1
  2. Item 2
  3. Item 3

Lista com letras começando de C

<ol type="A" start="3">
  1. Item 1
  2. Item 2
  3. Item 3

Código no editor:

   
    <ol type="A" start="3">
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
    </ol>

Lista com algarismos romanos começando do III

<ol type="I" start="3">
  1. Item 1
  2. Item 2

Código no editor:

    <ol type="I" start="3">
        <li>Item 1</li>
        <li>Item 2</li>
    </ol>

Listas não Ordenadas

<li>: Os itens da lista são representados por <li> (List Item - Item de Lista).

  • Maça

        <li>Uva</li>

    </ul></p>

Lista de Definições

As listas de definições em HTML são usadas para exibir termos e suas respectivas definições, funcionando como um glossário ou um conjunto de especificações. Elas são representadas pelas seguintes tags:

Estrutura:

<dl> (Lista de definições): Defina uma lista de definições.

<dt> (Termo de Definição): Define o termo a ser explicado.

<dd> (Descrição da Definição): Defina a descrição do termo.

ex:

HTML
Linguagem de marcação usada para estruturar páginas web.
CSS
Linguagem de estilo utilizada para definir o design das páginas web.
JavaScript
Linguagem de programação usada para adicionar interatividade às páginas web.

Código no Editor:

<dl>  
    <dt>HTML</dt>  
    <dd>Linguagem de marcação usada para estruturar páginas web.</dd>  

    <dt>CSS</dt>  
    <dd>Linguagem de estilo utilizada para definir o design das páginas web.</dd>  

    <dt>JavaScript</dt>  
    <dd>Linguagem de programação usada para adicionar interatividade às páginas web.</dd>  
</dl>
