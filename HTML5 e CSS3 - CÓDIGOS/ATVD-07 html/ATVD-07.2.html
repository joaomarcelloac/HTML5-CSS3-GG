Tabelas

O que é uma tabela em HTML?

Uma tabela em HTML é uma estrutura usada para organizar dados em linhas e colunas, semelhante a uma planilha. Ela é composta por células, que são os "blocos" onde os dados são inseridos. As tabelas são muito úteis para exibir informações tabulares, como listas de produtos, horários, dados financeiros, etc.

Estrutura básica de uma tabela

A estrutura básica de uma tabela em HTML é composta por três principais tags:

○ <table>: Define o início e o fim da tabela.

○<tr>: Define uma linha (table row) na tabela.

○<td>: Define uma célula de dados (table data) dentro de uma linha.

ex:

Linha 1, Coluna 1 Linha 1, Coluna 2
Linha 2, Coluna 1 Linha 2, Coluna 2

No editor de código: 

<table>
    <tr>
        <td>Linha 1, Coluna 1</td>
        <td>Linha 1, Coluna 2</td>
    </tr>
    <tr>
        <td>Linha 2, Coluna 1</td>
        <td>Linha 2, Coluna 2</td>
    </tr>
</table>

Cabeçalho da tabela

Para adicionar um cabeçalho à tabela, usamos a tag <th>. O texto dentro de <th> é geralmente exibido em negrito e centralizado.

ex:

Nome Idade Cidade
João 25 São Paulo
Maria 30 Rio de Janeiro

No editor de código:

<table>
    <tr>
      <th>Nome</th>
      <th>Idade</th>
      <th>Cidade</th>
    </tr>
    <tr>
      <td>João</td>
      <td>25</td>
      <td>São Paulo</td>
    </tr>
    <tr>
      <td>Maria</td>
      <td>30</td>
      <td>Rio de Janeiro</td>
    </tr>
</table>

colspan (Mesclar Colunas)

O atributo colspan é usado dentro das tags <th>ou <td>para fazer uma célula ocupar várias colunas.

Exemplo: Mesclando colunas para um título centralizado

Título da Tabela
Nome Idade Cidade
João 25 São Paulo

No editor de código:

    <table border="2">
      <tr>
          <th colspan="3">Título da Tabela</th>
      </tr>
      <tr>
          <th>Nome</th>
          <th>Idade</th>
          <th>Cidade</th>
      </tr>
      <tr>
          <td>João</td>
          <td>25</td>
          <td>São Paulo</td>
      </tr>
</table>

Explicação:

○ O colspan="3" faz com que a célula do título ocupe três colunas , centralizando a informação.

○ Isso é útil para criar cabeçalhos que se estendem por várias colunas.


width (Largura da Tabela ou Célula)

O atributo width define a largura da tabela inteira ou de uma célula individual.

Exemplo: Definindo a largura da tabela

Nome Idade Cidade
João 25 São Paulo

No editor de código:

    <table border="2" width="500">
    <tr>
        <th>Nome</th>
        <th>Idade</th>
        <th>Cidade</th>
    </tr>
    <tr>
        <td>João</td>
        <td>25</td>
        <td>São Paulo</td>
    </tr>
</table>

Explicação:

○ width="500"→ Defina que a largura total da tabela será de 500 pixels .

Exemplo: Definindo a largura de uma célula

Nome Idade Cidade
João 25 São Paulo

No editor de código:

    <table border="2">
    <tr>
        <th width="200">Nome</th>
        <th width="100">Idade</th>
        <th width="150">Cidade</th>
    </tr>
    <tr>
        <td>João</td>
        <td>25</td>
        <td>São Paulo</td>
    </tr>
</table>

Explicação:

○ A célula “Nome” terá 200 pixels de largura.

○ A célula “Idade” terá 100 pixels.

○ A célula “Cidade” terá 150 pixels.

height (Altura da Tabela ou Célula)

O atributo heightdefine a altura da tabela ou de células individuais.

Exemplo: Definindo a altura da tabela

Nome Idade
João 25

No editor de código:

    <table border="2" width="300" height="200">
    <tr>
        <th>Nome</th>
        <th>Idade</th>
    </tr>
    <tr>
        <td>João</td>
        <td>25</td>
    </tr>
</table>

Explicação:

○ height="200"→ A altura total da tabela será de 200 pixels .

Exemplo: Definindo a altura de uma célula

Nome Idade
João 25

No editor de código:

    <table border="2">
    <tr>
        <th height="50">Nome</th>
        <th height="50">Idade</th>
    </tr>
    <tr>
        <td>João</td>
        <td>25</td>
    </tr>
</table>

Explicação:

○ height="50"→ Defina que a altura das células da primeira linha será de 50 pixels .


OBS: Quando você tem uma tabela em HTML com células vazias (<td> </td>), o navegador pode tentar ajustar automaticamente o layout, o que pode fazer com que a tabela "colapse" ou perca a largura esperada. Para evitar isso, uma solução é adicionar um espaço não quebrável ( &nbsp; ) dentro das células vazias. Isso garante que a célula tenha um conteúdo, mesmo que seja invisível, evitando que o navegador ajuste o tamanho da tabela de forma inesperada.
