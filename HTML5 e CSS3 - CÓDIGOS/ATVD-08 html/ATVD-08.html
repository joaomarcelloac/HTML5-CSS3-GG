Usando links externos

A tag <a> em HTML é usada para cria++++r hiperlinks , permitindo que os usuários naveguem para outras páginas, contidas dentro da mesma página ou até façam o download de arquivos.

Estrutura básica do <a> (Âncora):

    <a href="https://www.exemplo.com">Clique aqui</a>

ex:

○Você pode acessar o meu repositório público no GitHub

○Você tambem pode acessar meu canal gratuíto no Youtube

Atributos Importantes para Links Externos

target="_blank": Abra o link em uma nova aba do navegador.

rel="noopener"(Segurança para target="_blank"): Evite vulnerabilidades em que o site externo possa manipular a página original.

rel="noreferrer"(Privacidade): O site de destino não obtém informações sobre de onde o usuário veio.

rel="nofollow"(SEO): ndica que o link não deve ser seguido pelos buscadores , usado para links pagos ou patrocinados .

rel="external"(Indica que o link é externo): Diz ao navegador que o link aponta para um site externo.

Dica: Use target="_blank" sempre com rel="noopener noreferrer" para manter segurança e privacidade


Abrir o conteúdo do link em uma nova aba ou janela, sem que o usuário saia do site atual

Por padrão, sempre que um visitante clica em um hyperlink, o site de destino abre na mesma janela do site que continha esse link. Ou seja, o conteúdo anterior vai deixar de ser exibido para mostrar o novo conteúdo.

Esse é um comportamento desejado quando o visitante vai continuar a visitar o nosso site, apenas mudando de um documento para o outro. Mas e quanddo um clique leva o visitante para o outro site e provavelmente ele nunca mais voltara ao nosso site?

Para poder controlar onde o site de destino vai abrir, podemos usar o atributo target, que suporta os seguintes valores:

_blank: Vai abrir o link em uma nova janela em branco.

_self: Vai abrir o link na janela ou frame atual (padrão).

_top: Vai desfazer todos os frames e abrir o destino no navegador completo.

_parent: Similar ao uso do _top em uma referência à janela mãe

nome-do-frame: Caso esteja usando frames, indicar o nome da janela a abrir.

ex:

Você pode acessar o meu repositório público no GitHub.

No editor de código:

    <p>Você pode acessar o meu <a href="#" target="_blank" rel="externos">repositório público no GitHub</a>.</p>

Atributo rel (relacionamento)

O atributo rel(relacionamento) no elemento <a> define o relacionamento entre a página atual e a página para a qual o link aponta. Ele ajuda na navegação, SEO e segurança.

Valores do atributo rel na tag <a>

next: Indica que o link é para a próxima parte do documento atual.

prev: Indica que o link é para a parte anterior do documento atual.

author: Indica que é um link para o site do autor do artigo atual.

external: Indica que é um link para outro site que não faz parte do site atual.

nofollow: Indica que é um link para um site não endossado, como um link pago.

alternate: Indica uma versão alternativa do documento atual (exemplo: outra língua ou formato).

bookmark: Define um link como um marcador dentro do documento.

help: Indica que o link leva para uma página de ajuda ou documentação.

license: Define um link que aponta para a licença do documento.

noopener: Impede que a nova página tenha acesso ao objeto window.opener, melhorando a segurança.

noreferrer: Evita que o navegador envie informações de referência (referrer) ao site de destino.

ugc: Indica que o link foi gerado por um usuário, como em comentários ou postagens de fóruns.

sponsored: Indica que o link é patrocinado, usado para evitar penalizações no SEO de links pagos.

Sintaxe básica:

Visite o site

No editor de código:

    <a href="https://exemplo.com" rel="noopener">Visite o site</a>

Usando links internos

Os links internos fazem a relação com outra página que está dentro do mesmo servidor. Eles conectam diferentes partes do site sem precisar usar URLs externos.

Atributos importantes para links internos

rel="next"→ Indica que a página linkada é a próxima parte do conteúdo.

rel="prev"→ Indica que a página linkada é a parte anterior do conteúdo.

rel="nofollow"→ Indica que os motores de busca não devem seguir esse link.


ex:

Esta é a primeira página do site. Se você quiser, pode acessar também a minha segunda página

Você também pode acessar a nossa página de noticias.

No editor de código:

    
Esta é a primeira página do site. Se você quiser, pode acessar também a minha <a href="PAG_INTERNA_1.html" target="_self" rel="next">segunda página</a>

    
Você também pode acessar a nossa <a href="NOTICIAS/PAG_INTERNA_2.html" rel="next" target="_self">página de notícias.</a>
